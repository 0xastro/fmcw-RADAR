◆ chirp0Idx

uint16_t MmwDemo_BpmCfg_t::chirp0Idx


If BPM is enabled, this is the chirp index for the first BPM chirp. It will have phase 0 on both TX antennas (+ +).

If BPM is disabled, a BPM disable command (set phase to zero) will be issued for the chirps in the range [chirp0Idx..chirp1Idx].

uint16_t MmwDemo_BpmCfg_t::chirp1Idx


If BPM is enabled, this is the chirp index for the second BPM chirp. It will have phase 0 on TX0 and phase 180 on TX1 (+ -).
