mmWave SDK Module Documentation

The mmWave software development kit (SDK) is a unified software platform for the TI mmWave Sensing Portfolio, providing easy setup and fast out-of-the-box access to evaluation and development. The module documentation is intended for software application developers using the mmWave SDK. This document includes links to detailed documentation exposed by each module in the package.
